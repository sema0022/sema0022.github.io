Bootstrap This!

What is Bootstrap?

Bootstrap is a popular front-end framework that can be used to easily build responsive, mobile-first websites. Open-source, it boasts a vast library of components and numerous predefined classes to help you create quickly and efficiently.

Benefits of Bootstrap

Responsive Layouts

Bootstrap is a responsive framework based on a twelve-column grid system that uses flexbox. With a simple structure of containers, rows, and columns, it allows you to design mobile-first layouts with ease. In fact, there is no need stress over media queries, as Bootstrap provides six responsive breakpoints that can be applied directly to your grid layout.

Predefined Classes

Customize your website’s design using Bootstrap’s predefined classes. No separate CSS stylesheet is required when you can adjust colour, text, spacing, and other elements just by adding the appropriate classes to your HTML.

Prebuilt Components

There is no need for additional CSS and JavaScript when you have Bootstrap’s prebuilt components. Add alerts, badges, buttons, and more to your website by simply referencing their corresponding classes in your HTML. It’s really that easy.

Getting Started

Not sure where to begin? Copy and paste Bootstrap's Starter Template into a new HTML document. Then, take a look through Bootstrap's Documentation Page to discover all the different ways you can build your site! Remember, Bootstrap is based on a 12-column grid system; a basic structure will consist of a container, with a row inside, which in turn contains columns.

Featured Components

Accordions

Using the accordion component allows you to vertically collapse content and minimize the space it takes up on the page.

Buttons

Provide buttons to users to encourage actions from them. You can easily change the style and state to convey your intended message.

Alerts

Is there something your users need to know? Alert them with a message on their screens, and give it a colour according to your intended context.

Popovers

Popovers are little pop-up messages that are triggered by a user’s action, like clicking a button. To use them, be sure to uncomment the “Bootstrap Bundle with Popper” option in your Starter Template.

Spinners

Spinners are a neat way to visually display a loading state. Bootstrap gives you the option to use either a border spinner or a growing spinner.
